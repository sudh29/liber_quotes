# Daily Quotes

A small React/Vite app that shows one curated quote each day.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## App Surfaces

- Daily Quotes app: `http://localhost:3000/`

## Source Layout

- `src/apps/daily-quotes/` - Daily Quotes experience, quote data, types, and hooks
